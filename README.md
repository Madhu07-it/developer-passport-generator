# 🚀 Developer Passport Generator

A creative Flask web application that generates a modern Developer Passport using user information, skills, and social media links.

## ✨ Features

* Modern Landing Page
* Developer Passport Generator
* Profile Image Support
* Skill Tags
* LinkedIn & GitHub Links
* Responsive Design
* Flask Backend
* Beautiful UI

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/developer-passport-generator.git
```

Navigate to the project folder:

```bash
cd developer-passport-generator
```

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```

## 📸 How It Works

1. Open the application.
2. Click "Create My Passport".
3. Enter your details.
4. Generate your Developer Passport.
5. Share your professional identity.
